# Chrome Web Store — Screenshot Capture List

Chrome Web Store wants **5 screenshots** at exactly **1280×800 pixels**
(landscape) OR **640×400** for smaller listings. Use **1280×800** —
the bigger size gets better placement and looks sharper on retina
displays. The first screenshot is what users see in search results,
so it has to sell the extension on its own.

You'll need a fresh Chrome window with the extension loaded. Set the
window to roughly 1280×800 first via the OS resize handle, or use
Chrome DevTools' device toolbar set to a custom 1280×800 size.

This is the one that has to sell the extension at a glance. Pick the
most visually clean tweet on Crypto Twitter right now.

## Practical tips

- **First screenshot is everything.** It's what shows up in search
  results. Make it the cleanest, most obvious win.
- **No emoji-overload.** Chrome's reviewers tolerate one or two but
  not a row of them.
- **No copyrighted logos in screenshots.** X's logo, Reddit's logo
  appearing in the UI is fine because that's the product context, but
  don't add Twitter/Reddit branding in your captions or annotations.
- **Sharp text.** Use 1280×800 native — don't upscale 640×400.
- **Real data.** Don't fake a screenshot — Chrome reviewers spot it.
  Use real tokens, real tweets, real Reddit posts.
- **Same browser theme across all 5.** Pick dark or light and stick
  with it. Mixing is jarring.

## Where to upload them

Chrome Web Store dashboard → your extension → "Store listing" →
"Screenshots" → upload each PNG/JPG in order. The order in the
dashboard is the order they appear in the listing. Lead with #1.
